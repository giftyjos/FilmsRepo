>git init [to initialize git]

>git status [to list all untracked files]

>git add -A [to adding all untracked files to staging area]

>git commit -m "commit message" [to commit added file to local respository] 

>git clone[to take exact copy from remote respository to local workspace]

>git pull[to take  changes from remote respository to local workspace]

>git push [to sent changes from local respository to remote respository ]

>git config --list [to list configuration]

>git config --global user.name "username" [to add username]

>>git config --global user.email "useremail"[to add useemail]

>git remote add remote_name remote_path [to add remote]

>git push remote_name branch
